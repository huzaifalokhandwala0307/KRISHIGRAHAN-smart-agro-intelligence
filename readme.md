# 🌱 KRISHIGRAHAN – Smart Agro Intelligence

An AI-powered web application that recommends the best **crop** and **fertilizer** based on soil and environmental conditions.

Built using **Machine Learning + Flask**, this project helps farmers make smarter agricultural decisions.

---

## 🚀 Features

- 🌾 Crop Recommendation using ML model
- 🧪 Fertilizer Prediction system
- 🌦️ Soil & climate-based analysis
- 📊 Inputs: N, P, K, temperature, humidity, pH, rainfall, moisture, soil type
- 📋 Cultivation advice for selected crops
- 💻 Clean and modern UI (custom CSS design)

---

## 🧠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Flask (Python)
- **Machine Learning:** Scikit-learn
- **Data Processing:** Pandas, NumPy
- **Model Storage:** Joblib

```bash
cd krishigrahan
```

---


---

```bash

```

---

## 🌐 Usage

1. Enter soil and environmental parameters:
   - Nitrogen (N)
   - Phosphorus (P)
   - Potassium (K)
   - Temperature
   - Humidity
   - pH
   - Rainfall
   - Moisture
   - Soil Type

2. Click **"Get Recommendation"**

3. The system will output:
   - 🌱 Recommended Crop
   - 🧪 Suggested Fertilizer
   - 📋 Cultivation Advice
